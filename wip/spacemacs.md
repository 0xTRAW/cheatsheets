### Layers

| Shortcut  | Description       |
| ---       | ---               |
| `SPC` `f` | File              |
| `SPC` `p` | Project           |
| `SPC` `t` | Toggle            |
| `SPC` `b` | Buffer            |
| ---       | ---               |
| `SPC` `m` | Major mode        |
| `,`       | Same as `SPC` `m` |
| ---       | ---               |
| `SPC` `g` | Git               |
| `SPC` `h` | Help              |
{: .-shortcuts}

| Shortcut        | Description    |
| ---             | ---            |

| ---             | ---            |
| `SPC` `h` `l`   | Layers         |

### File (`f`)

### Buffer (`b`)

| Shortcut      | Description                |
| ---           | ---                        |
| `SPC` `b` `n` | Next buffer (`:bnext`)     |

{: .-shortcuts}
