### Layers

| Shortcut  | Description       |
| ---       | ---               |
| `SPC` `f` | File              |
| `SPC` `p` | Project           |
| `SPC` `t` | Toggle            |
| `SPC` `b` | Buffer            |

| `SPC` `m` | Major mode        |

| ---       | ---               |
| `SPC` `g` | Git               |
